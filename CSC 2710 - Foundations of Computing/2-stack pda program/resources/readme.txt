To compile: enter working directory and type "javac *.java"

To run:
- enter working directory and type "java PDAParser" with no command-line arguments
- when prompted for input file, enter "filename.txt"
- (if the input file is not in a valid format, the program will end)
- when prompted, enter the string that you want to parse (ex: aabbbddd)
- press S or F for a step-by-step or fast run of the parse
- in the step-by-step run, press Enter to continue through the parse after viewing the current configuration
- press Y or N depending on whether or not you want to parse another string using the PDA
- if you press Y, enter a new string to parse
- if you press N, choose between Y or N again depending on whether or not you want to input a new PDA file
- if you press Y, start over from step 2
- if you press N, the program ends
